Engineering Mathematics

Mathematical methods for solving engineering problems including calculus, linear algebra, and differential equations

Topics

  • Differential and integral calculus with engineering applications. Key concepts include limits, derivatives, integrals, and their applications in engineering problems.

    Key Formulas:

    • Derivative: f'(x) = lim(h→0) [f(x+h)-f(x)]/h
    • Integral: ∫f(x)dx = F(x) + C

  • Vectors, matrices, and systems of linear equations. Applications include solving systems of equations, transformations, and vector spaces.

  • First and second order differential equations with engineering applications. Includes ordinary and partial differential equations.

Practice Problems

Problem 1: Find the derivative of f(x) = 3x² + 2x - 5

Solution: f'(x) = 6x + 2

Explanation: Apply the power rule to each term: d/dx(xⁿ) = nxⁿ⁻¹

Problem 2: Solve the differential equation dy/dx = 2x

Solution: y = x² + C

Explanation: Integrate both sides with respect to x: ∫dy = ∫2x dx
